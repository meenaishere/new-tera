Backend Setup

The backend logic for this application is located in the api/index.js file and needs to be deployed as a serverless function on a platform like Vercel or Netlify.

Deployment Steps:

  1. Deploy the project to a serverless hosting provider (e.g., Vercel, Netlify).
  2. Once deployed, you will get a URL for your serverless function (e.g., https://your-project-name.vercel.app/api).
  3. Update the API_ENDPOINT variable in the script of this HTML file to your serverless function URL.

The current backend code in api/index.js uses an up-to-date scraping method. However, serverless platforms are often blocked by Terabox. If you encounter an "IP Blocked" error, you may need to redeploy the function to get a new IP address.
